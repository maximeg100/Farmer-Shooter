# Farmer-Shooter
Jeu Unity 3D où un fermier peut tirer des projectiles (pizzas) sur des animaux lui courant dessus. Si les animaux passent derrière lui, le joueur perd la partie.
Les animaux apparaissent avec un délai entre chaque animal fixe. La nature de l'animal est aléatoire et chaque animal a sa propre vitesse.
Ce jeu a été fait en suivant le tutoriel sur le site officiel : https://learn.unity.com/course/create-with-code/unit/unit-2-basic-gameplay?version=6.0
